Nordea ID -sovellus

Nordea ID on mobiilisovellus, jolla voit kirjautua Nordean netti- ja mobiilipankkiin, vahvistaa tapahtumia sekä tunnistautua muiden palveluntarjoajien verkkopalveluihin.

Lue lisää Nordea ID -sovelluksesta
Nordea ID

Kirjautumistunniste

Tunnisteen avulla tiedät, että olet vahvistamassa oikeaa kirjautumista. Tarkista, että Nordea ID -sovelluksessa näkyy sama kirjautumistunniste kuin tällä sivulla.

Jos kirjautumistunnisteet eivät täsmää, älä hyväksy vahvistuspyyntöä.

Käyttäjätunnus

Käyttäjätunnus on Nordean sinulle myöntämä tunnus, jossa on 5-13 numeroa.

Nordea ID -sovellus QR-koodi

Kertakäyttöisen QR-koodin avulla pääset kirjautumaan syöttämättä käyttäjätunnustasi. QR-koodia käytetään yhdessä Nordea ID -sovelluksen kanssa.

  1. Napauta OK, niin saat QR-koodin näkyviin.
  2. Avaa Nordea ID -sovellus puhelimella tai tabletilla.
  3. Napauta sovelluksesta "Lue QR-koodi", joka avaa kameranäkymän QR-koodin lukemiseen.
  4. Kohdista kamera QR-koodiin. Varmista, että koko koodi näkyy sovelluksen kameranäkymän sisällä.
  5. Vahvista kirjautuminen PIN-koodilla tai biometrisellä tunnistautumisella (kuten kasvojen tunnistus tai sormenjäljen luku).

Puhuva tunnus­luku­laite

Puhuva tunnuslukulaite on vaihtoehto tunnistautumiseen asiakkaillemme, joilla ei ole älypuhelinta tai tablettia, johon saisi asennettua tunnuslukusovelluksen.

Nordea ID -laite ja puhuva tunnuslukulaite

Käyttäjätunnus

Käyttäjätunnus on Nordean sinulle myöntämä tunnus, jossa on 5-13 numeroa.

Tunnusluku

Paina puhuvan tunnuslukulaitteen LOGIN-painiketta ja seuraa ohjeita saadaksesi kirjautumiseen tarvittavan tunnusluvun.

Nordea ID -laite

Nordea ID -laite on laite, jolla voit kirjautua Nordean netti- ja mobiilipankkiin ja vahvistaa tapahtumia. Laitteessa on kamera, jolla luetaan tunnistuspalvelussa näkyvä QR-koodi.

Käyttäjätunnus

Käyttäjätunnus, jonka olet saanut Nordealta. Se muodostuu 5–13 numerosta.

Kirjautumistunniste

Tunnisteen avulla tiedät, että olet vahvistamassa oikeaa kirjautumista. Tarkista, että Nordea ID -laitteessa näkyy sama kirjautumistunniste kuin tällä sivulla.

Jos kirjautumistunnisteet eivät täsmää, älä hyväksy vahvistuspyyntöä.

Kirjaudu / vahvista tapahtuma Nordea ID -laitteella

  1. Valitse kirjautumistavaksi "Nordea ID -laite".
  2. Syötä käyttäjätunnuksesi ja napauta OK. QR-koodi ilmestyy näytölle.

Vahvistustunnus

Vahvistustunnuksessa on yhdeksän numeroa. Se näytetään laitteessasi, kun olet lukenut QR -koodin.

Kirjautumistunniste

Tunnisteen avulla tiedät, että olet vahvistamassa oikeaa kirjautumista. Tarkista, että Nordea ID -laitteessa näkyy sama kirjautumistunniste kuin tällä sivulla.

Jos kirjautumistunnisteet eivät täsmää, älä hyväksy vahvistuspyyntöä.

Kirjaudu / vahvista tapahtuma Nordea ID -laitteella

  1. Paina Nordea ID -laitteen SCAN-painiketta.
  2. Kohdista kamera QR-koodiin. Varmista, että koko koodi näkyy laitteen kameranäkymän sisällä.
  3. Varmista, että laitteessa lukeva teksti vastaa sitä mitä haluat tehdä. Paina OK.
  4. Syötä laitteeseen PIN-koodisi ja paina OK.
  5. Laite näyttää vahvistustunnuksen. Syötä se vahvistustunnus -kenttään.
